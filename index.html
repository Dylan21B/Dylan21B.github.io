Project Overveiw

I want to build an application that will replace the current delivery system at my work (Ace Hardware). The current procedure for a delivery is to fill out all the data on a delivery slip and then place it in a stack in the office. I want to modernize this procedure and eliminate some of the room for human error. This will prevent slips from getting lost, or forgetting about the delivery completely! (This happened once.) By using the app, the user will have all info displayed in an organized manner with data readily available. They will be able to create and update deliveries on the fly. Simplicity of the app will be one of my main focuses.
